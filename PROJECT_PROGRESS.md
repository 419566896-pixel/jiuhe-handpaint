## 部署说明

### 本地运行
```bash
cd /Users/mac/Projects/jiuhe-handpaint
npm run dev
# 访问 http://localhost:3000
```

### 部署到 Vercel（免费）
```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 登录
vercel login

# 3. 部署
cd /Users/mac/Projects/jiuhe-handpaint
vercel

# 4. 生产部署
vercel --prod
```

或者：
1. 推送代码到 GitHub
2. 在 https://vercel.com 导入仓库
3. 一键部署（自动检测 Next.js）
