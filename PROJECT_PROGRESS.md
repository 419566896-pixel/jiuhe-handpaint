## 经验总结

1. **图片压缩优先** — 140MB 的原始图片会严重影响加载速度
2. **自动化测试** — Vitest 配合 Testing Library 很好用
3. **npm run check** — lint+test+build 一键检查，避免低级错误
4. **SEO 要趁早** — JSON-LD/OG/alt 文本在开发阶段就做好
5. **AI 可见性** — robots.txt 显式允许 AI 爬虫 + llms.txt 是新趋势
6. **小程序复用** — 官网图片和文案可以直接复用到小程序
